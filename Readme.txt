【サンプル内容】
本サンプルは「アンリアルクエスト2～グレイマンからの5つのオーダー～」の参加者に向けて Unreal Engine 4.26.2 で制作されたサンプルです。
シンプルな形状で組まれたステージとゲームとしての最小限の要素が用意されています。各クエストをこなしてリッチにしていきましょう！

【フォルダ構成】
ContentBrowser
    Splash：プロジェクト起動時のスプラッシュ画像が格納されています。操作等は不要です。
    UnrealQuest2
        Animations：キャラクターアニメーション、アニメーションBPを格納しています。
        Blueprints：アンリアルクエスト用のGameMode、キャラクターBP、アイテムBP、ゴールBPが格納されています。
        Maps：シンプルな形状で作成されたレベルが格納されています。
        Materials：ステージ用のマテリアルを格納しています。
        Meshes：ステージにて使用されている各種モデルデータを格納しています。

【各アセットの役割】
ブループリント
    BP_GameMode：ゲームの開始、スコアの加算処理が用意されています。
    BP_Player：Unreal Engine のThirdPersonTemplateに付属しているBPにスコアの加算処理、被ダメージ処理を追加しています。
    BP_Item：プレイヤーキャラクターが触れるとスコアが加算されるブループリントです。
    BP_Goal：プレイヤーキャラクターが触れるとゴール処理が行われるブループリントです。
    BP_DamageArea：プレイヤーキャラクターが触れるとダメージを与えるブループリントです。
    ThirdPerson_AnimBP：Unreal Engine のThirdPersonTemplateに付属しているアニメーションBPです。

レベル(マップ)
    GrayBox：シンプルなボックス等の最小限の内容で構成されたゲームステージです。こちらのレベルを元にリッチにしていきましょう！
